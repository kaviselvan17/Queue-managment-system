# Queue-managment-system

📌 Project Overview

QLess is a simple web-based Smart Queue Management System designed to reduce waiting time and improve customer experience. Users can join a queue, receive a token number, and view live queue information. Administrators can log in to manage and reset the queue.

This project is built using HTML, CSS, and JavaScript and uses the browser's Local Storage to store queue data.

Features:
    User Features
    Join the queue by entering:
    Name
    Phone Number
    Service Type
    Generate unique token numbers automatically.
View:
    Assigned token number
    Queue confirmation message
    Number of people ahead
    Estimated waiting time
    Live progress bar indicating queue status.
    Admin Features
    Secure admin login.
    Access admin dashboard.
    Reset the entire queue system.
    Manage queue information.
Technologies Used:
    HTML5
    CSS3
    JavaScript (Vanilla JS)
    Local Storage API
